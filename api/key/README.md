因为是通过即时查询百度结果,大概每页要1s左右的时长,具体视服务器性能和带宽影响

所以查询的时候要注意关键词的数量

例如要查询前5页的排名情况

每个词要查5页,每页1s

每个词大概要5s,如果服务器性能好,每词可以大概到1s的时长,具体可以自行测试.
